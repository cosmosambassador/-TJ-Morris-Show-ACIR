## GitHub Pages setup

To publish for free:

1. Open this repository on GitHub.
2. Go to **Settings**.
3. Open **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch: **main**.
6. Select folder: **/(root)**.
7. Save.

The free GitHub Pages URL should look similar to:

`https://cosmosambassador.github.io/-TJ-Morris-Show-ACIR/`

If a cleaner repo is later created as `tjmorrisshow`, the cleaner Pages address would be:

`https://cosmosambassador.github.io/tjmorrisshow/`
